# Recap Project 6: Tourio App

**Tourio** is the friend of all travelers: it displays sights you want to visit and lets you edit them.

## Tech Stack

- JavaScript
- React
- Next.js
- MangoDB Atlas
- Mangoose
- Styled Components
- useSWR
- HTML
- CSS
